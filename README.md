## Installation

1. `git clone https://github.com/syedoz/reactitunes.git`
2. `cd reactitunes`
3. `npm install`



## Dev

Run webpack dev server

```
npm run start
```

and go to `localhost:8080`

## Deploy

Use webpack minify plugin

```
npm run build
```
